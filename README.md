# Stock Portfolio Optimization System

## Overview
A Python-based portfolio optimization tool that determines the best stock allocation under budget and risk constraints.

## Features
- Portfolio optimization
- Budget constraint checking
- Risk constraint validation
- Expected return calculation
- Automatic selection of optimal stock combinations
- Output report generation

## Technologies Used
- Python
- itertools
- Algorithm Design

## Algorithm
The program evaluates possible stock combinations and selects the portfolio with the highest expected return while satisfying budget and risk constraints.
